Medidas, Colores, Fondos, Fuentes e Iconos

Unidades de Medidas

Absolutas

Son medidas fijas que tienen su propio valor y no dependen de otro elemento, son muy poco flexibles.

Relativas

Son medidas que dependen de otro elemento, son muy flexibles.

Unidades de Medidas Absolutas

Son medidas muy poco flexibles y no se adaptan fácilmente a los diferentes dispositivos.

Pixel

Es la unidad de medida más pequeña y su medida real depende del dispositivo.

CM

Centímetros (10mm).

MM

Milímetros (1/10cm).

IN

Pulgadas (2.54cm).

PT

Puntos (0.35mm).

PC

Picas (4.23mm).

Unidades de Medidas Relativas

Su valor depende de otro valor, son muy flexibles y son las mas utilizadas.

em

< < M> > 1em = Tamaño de fuente establecida en el navegador, si no es modificada por el usuario su valor base es de 16px, osea que 1em = 16px, 2em = 32px, 0.5em = 8px, etc.

rem

< < root M> > 1rem = tamaño fuente raíz, toma la medida base que el usuario le da con la pseudo clase :root, si el usuario configura en 22px, 1rem = 22px, 2rem = 44px, etc.

%

Porcentaje, relativo al contenedor padre.

ex

< < X> > (0.5 em aproximadamente) 1ex = mitad del tamaño de la fuente del navegador aproximadamente.

ch

< < 0 width> > 1ch = ancho de un cero.

Unidades de medidas flexibles

vh

Viewport Height: si el div o elemento padre mide 800px de alto, la medida seria 50vh = 400px de alto.

vw

Viewport Width: si el div o elemento padre mide 800px de ancho, la medida seria 50vw = 400px de ancho.

vmin

Viewport Minimum

vmax

Viewport Maximum

Colores y Tipografías

Colores

Para dar color a cualquier elemento se utiliza la propiedad color, existen diferentes formas de especificar el color.

Hexadecimal

#000000

RGB

red, green, blue = rgb(0, 0, 0)

RGBA

red, green, blue, alpha(opacidad) = rgba(0, 0, 0, 1)

HSL

hue(Matiz), saturation(saturación), lightness(luminosidad) = hsl(0, 0%, 0%)

HSLA

hue(Matiz), saturation(saturación), lightness(luminosidad), alpha(opacidad) = hsla(0, 0%, 0%, 1)

Color de Fondo

Para dar color de fondo a cualquier elemento se utiliza la propiedad background-color, existen diferentes formas de especificar el color.

Imagen de Fondo

Para dar una imagen de fondo a cualquier elemento se utiliza la propiedad background-image, existen diferentes formas de especificar la imagen.

Tipografías

Es uno de los pilares del diseño web, se pueden dar diferentes estilos y es muy importante saber interpretar al usuario o diseño para elegir la tipografía correcta.

Propiedades básicas:
  • (Tipo de fuente)font-family: Arial, Helvetica, sans-serif;
  • (Tamaño)font-size: 16px;
  • (Grosor)font-weight: bold;
  • (Estilo)font-style: italic;
  • (Decorado)text-decoration: underline;
  • (Transformación)text-transform: uppercase;
  • (Alineado)text-align: center;
  • (Altura)line-height: 1.5;
  • (Espacio entre letras)letter-spacing: 1px;
  • (Espacio entre palabras)word-spacing: 1px;

Google Fonts

Es una librería de fuentes de google, se pueden utilizar en cualquier proyecto web.

¿Cómo se utiliza?

Se debe importar la fuente en el archivo css, copiando el link que nos da google fonts.

Se debe agregar la propiedad font-family: 'nombre de la fuente'; al elemento que queremos cambiar la tipografía.

Iconos

Los iconos son muy importantes en el diseño web, ya que ayudan a mejorar la experiencia de usuario y a darle un toque de diseño a la página.

¿Cómo se utilizan?

Se debe importar la librería de iconos en el archivo html, copiando el link que nos da la librería.

Se debe agregar la etiqueta < < i> > con la clase del icono que queremos utilizar.
